      ``` [Python]
	def quicksort(l):  
	'''Ordena arreglo, L = arreglo, (L)-->(L Ordenado)'''  
	if len(l) <= 1:  
		return l  
	else:  
		pivote = l[len(l)//2]  
		igual = [i for i in l if i==pivote]  
		menor = [i for i in l if i<pivote]  
		mayor = [i for i in l if i>pivote]  
	return quicksort(menor)+igual+quicksort(mayor)
   ```
